# NewsProject 📰

An Android news application that fetches and displays the latest news articles 
from around the world using a REST API.

## Tech Stack
- **Language:** Kotlin
- **IDE:** Android Studio
- **Networking:** Retrofit (REST API)
- **Version Control:** Git

## Features
- Browse latest news articles by category
- View full article details
- Clean and intuitive UI

## Getting Started

1. Clone the repository
```bash
   git clone https://github.com/eliasm98/NewsProject.git
```
2. Open the project in **Android Studio**
3. Build and run on an emulator or physical device
